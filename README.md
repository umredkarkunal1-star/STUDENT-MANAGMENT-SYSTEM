# 🎓 Vidya Bharati Student Management System

A modern and responsive web-based Student Management System designed to streamline school operations. The platform provides role-based access for Admins, Teachers, and Students, enabling efficient management of academics, attendance, fees, reports, and analytics.

## ✨ Features

* 🔐 Role-Based Authentication (Admin, Teacher, Student)
* 👨‍🎓 Student Management
* 👨‍🏫 Teacher Management
* 📅 Attendance Tracking
* 💰 Fee Management & Receipts
* 📚 Subject Management
* 📊 Analytics Dashboard
* 📈 Reports & Exports
* 📂 File Upload System
* 🔔 Notifications Panel
* 🤖 School Assistant Chatbot
* 🌙 Light/Dark Theme Support
* 📱 Fully Responsive Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Local Storage

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/vidya-bharati-sms.git
```

2. Open the project folder

3. Run using Live Server or open `index.html` in your browser.
